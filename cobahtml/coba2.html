Untuk Apa HTML?

Fondasi utama dalam pengembangan website adalah menguasai html dengan kuat. Jika kita ibaratkan html seperti anatomi tubuh hewan maka html menjadi kerangka tulangnya

Elemen HTML yang perlu anda ketahui

title : untuk menulis judul h1 : untuk heading utama
h2 : untuk subheading h1
h3 : untuk subheading h2
h seterusnya hingga h6
p : paragraf digunakan untuk menuliskan konten
